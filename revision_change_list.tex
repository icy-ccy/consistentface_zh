\section*{一、参考写法提炼}

本轮修订只参考 \texttt{2501.05177v2.pdf} 的论文写法，不借用其技术内容。参考论文的主要写法特征如下：
\begin{itemize}
  \item 摘要使用单段紧凑写法，直接进入任务难点，随后说明现有方法限制，再给出方法、关键模块和实验结论。
  \item 绪论采用线性推进：任务背景、现有路线不足、本文方法一句话、三项挑战、对应解决策略。
  \item 方法开头先说明整体管线和各模块职责，再进入公式细节。
  \item 实验部分先说明每组实验回答什么问题，再解释设置、结果和结论。
  \item 结论部分明确适用边界，不把恢复结果表述为身份真实性证明。
\end{itemize}

\section*{二、正文具体修改}

\begin{enumerate}
  \item \textbf{摘要重写}
  \begin{itemize}
    \item 位置：\texttt{consistentface\_tip\_zh.tex} 摘要部分。
    \item 修改：改为单段紧凑摘要，按“任务难点-现有问题-ConsistentFace 方法-训练/推理边界-实验结论”的顺序组织。
    \item 结果：摘要更接近参考论文的直入式写法，减少背景铺垫、分段停顿和重复解释。
  \end{itemize}

  \item \textbf{绪论大修}
  \begin{itemize}
    \item 位置：\texttt{consistentface\_tip\_zh.tex} 绪论。
    \item 修改：继续参考 FaceMe 的开头推进方式，将绪论组织为“人脸恢复病态性-现有两类路线-外部条件可信度缺口-本文方法-三项挑战-对应设计”。
    \item 结果：读者能更快理解本文不是单纯增加参考或文本条件，而是解决参考与文本先验的可信度分配问题。
  \end{itemize}

  \item \textbf{贡献列表收紧}
  \begin{itemize}
    \item 位置：绪论末尾贡献列表。
    \item 修改：改为四点贡献，分别对应冲突感知双先验问题定义、多维可信度证据、属性槽级路由与回退机制、冲突监督和评测协议。
    \item 结果：贡献项与修改计划保持一致，也更像审稿人可快速扫描的条目。
  \end{itemize}

  \item \textbf{相关工作继续润色}
  \begin{itemize}
    \item 位置：\texttt{相关工作} 三个小节。
    \item 修改：按“研究路线-已有局限-本文差异”的结构重写，强化与 FaceMe、CoFaDiff、IConFace、MCS、A$^2$BFR 以及 IP-Adapter、PhotoMaker、InstantID、PuLID 的边界。
    \item 结果：相关工作从文献罗列改为问题导向叙述，更清楚地区分“更强先验注入”和“可信度分配”。
  \end{itemize}

  \item \textbf{结论性语言核查与补引}
  \begin{itemize}
    \item 位置：摘要、绪论、相关工作、主结果分析、强双先验基线、局限性和伦理说明。
    \item 修改：将“证明”“不可替代”“不能替代”等绝对表述降调为“表明”“在本文设置下有必要”“在本文冲突设置下不能替代”等范围限定表达。
    \item 补引：新增图像描述/视觉语言模型幻觉、面部识别用途和风险相关文献，并在正文对应位置引用。
    \item 结果：结论性表述更符合外审要求，外部领域判断由文献支撑，本文方法收益由本文实验链支撑。
  \end{itemize}

  \item \textbf{问题表述重构}
  \begin{itemize}
    \item 位置：\texttt{问题表述}。
    \item 修改：先定义输入输出，再说明三类不确定性来源；随后解释三类路由决策和训练/推理标签边界。
    \item 结果：符号、任务和标签使用关系更清楚，避免读者误以为推理阶段需要冲突标签。
  \end{itemize}

  \item \textbf{退化观测可靠锚点表述统一}
  \begin{itemize}
    \item 位置：摘要、绪论、问题表述、方法证据定义、反事实实验分析、自然语言提示分析、局限性和结论。
    \item 修改：统一说明退化图像是目标人脸的可靠观测锚点；退化强度只表示从该观测中可提取的身份/属性证据变弱，不表示观测图像本身不可靠。
    \item 结果：全文将不可靠来源限定为参考图像和文本提示等外部先验，并明确 $s_{yR}$、$s_{yT}$、$s_{ya}$ 是以可靠退化观测为锚点评估参考/文本是否偏离目标观测。
  \end{itemize}

  \item \textbf{方法开场改写}
  \begin{itemize}
    \item 位置：\texttt{方法} 开头。
    \item 修改：先说明 SDXL/ControlNet、参考 token、文本 token 的来源，再用三步概括 ConsistentFace 的新增模块。
    \item 结果：方法部分更像参考论文的“先给整体管线，再进入模块细节”的写法。
  \end{itemize}

  \item \textbf{方法部分按 FaceMe 式写法二次大修}
  \begin{itemize}
    \item 位置：\texttt{方法} 全节，包括条件编码、多维可信度证据、全局路由、属性槽路由、训练目标与推理策略。
    \item 修改：重写方法叙述顺序，改为“整体框架-模块职责-具体公式-训练/推理边界”的推进方式；减少逐符号流水账解释，增加每个模块为何存在、解决什么冲突和如何与恢复主干衔接的说明。
    \item 结果：方法部分更接近 FaceMe 的方法写法节奏，但未借用其技术内容；公式标签、核心方法主张和实验数值保持不变。
  \end{itemize}

  \item \textbf{实验设置改写}
  \begin{itemize}
    \item 位置：\texttt{评测设置} 和 \texttt{数据构建和划分}。
    \item 修改：实验开头改为两个核心问题；数据构建段落改为“固定变量-冲突来源-反事实划分-标签隔离”的顺序。
    \item 结果：实验目的和公平性更明确。
  \end{itemize}

  \item \textbf{结果分析改为观察式写法}
  \begin{itemize}
    \item 位置：五种子同一主干结果、强双先验基线、属性槽消融、错误来源区分、更换评估器、自然语言提示等小节。
    \item 修改：每个小节先说明它回答的问题，再给数值结果和解释；主结果表后改为“三点观察”。
    \item 结果：实验叙述更有层次，减少流水账式报告。
  \end{itemize}

  \item \textbf{局限性和结论重写}
  \begin{itemize}
    \item 位置：\texttt{局限性和结论}。
    \item 修改：明确 ConsistentFace 是先验使用可靠性方法，不是身份认证系统；保留 44.6\% 排名反转这一负面结果，并解释失败来源。
    \item 结果：结论更审慎，适用边界和伦理约束更明确。
  \end{itemize}
\end{enumerate}
